# Library Management System

A full-stack Library Management application built with Angular 16+ and ASP.NET Core 7+, featuring JWT authentication, reactive state management, and CRUD operations for library issues.

# Features

User Authentication: JWT-based login/register, role management

Issue Management: Create, read, update, delete library issues

File Uploads: Upload member photos or issue-related files

Reactive Frontend: Angular signals for live updates

Form Validation: User-friendly messages using a ValidationHelper

Secure API: TokenInterceptor automatically attaches JWT

# Tech Stack

Frontend: Angular 16+, TypeScript, Signals, RxJS

Backend: ASP.NET Core 7+, EF Core, Identity, JWT Authentication

Database: SQL Server (or any EF Core supported DB)
